# RedBlink VS Code Extension

VS Code extension component of RedBlink AI debugging assistant.

## Structure

- `src/` - Source code
  - `extension.ts` - Main entry point
  - `errorDetector/` - Error detection service
  - `questionGenerator/` - Question generation
  - `aiProvider/` - AI model integrations
  - `ui/` - Webview sidebar
  - `utils/` - Utility functions
- `test/` - Test files
- `package.json` - Extension manifest
- `tsconfig.json` - TypeScript configuration
