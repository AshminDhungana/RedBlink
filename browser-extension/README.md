# RedBlink Browser Extension

Browser extension component of RedBlink AI debugging assistant.

Supports:
- Chrome
- Firefox
- Edge
- Brave

## Structure

- `src/` - Source code
  - `background/` - Service worker
  - `content/` - Page injection script
  - `popup/` - Extension popup UI
  - `shared/` - Shared code with VS Code
- `test/` - Test files
- `manifest.json` - Extension manifest
- `icons/` - Extension icons
- `dist/` - Built extension
